# FareCalculator
This is a simple console-based cab fare calculator built using Python. It calculates the ride cost based on distance, car type, and peak-hour pricing.

Features
* Enter distance in kilometers
* Choose car type (Economy, Premium, SUV)
* Calculate fare based on per km rate
* Apply surge pricing during peak hours (5 PM – 8 PM)
* Display total fare

Tools Used
Python

How to Run
Run: python farecalculator.py
